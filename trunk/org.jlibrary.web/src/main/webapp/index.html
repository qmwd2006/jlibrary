About jLibrary demo

This demo shows a dynamic web application developed with jLibrary. You will be able to connect to the demo repository through the web application and browse it, modify it, search it, download content or even obtain RSS feeds to be notified about updates in the repository content.

By default you will be connected to the repository with a guest user so you do not need to log in to work with the demo. However, you can user the following preconfigured users: reader/reader that only has reader permissions (effectively will behave as the guest user) and editor/editor that has editor permissions.

The special thing about this demo is that you can also connect to the demo repository from your jLibrary desktop application using the url http://www.jlibrary.org/demo. You can use the same login information and work from your desktop and the changes will be immediately visible from the web application!
